Release checklist

1) Ensure Stream Deck CLI is installed:
   streamdeck -v

2) Validate the plugin:
   streamdeck validate com.pillowfresco.birdnetmqtt.sdPlugin

3) Package the plugin:
   streamdeck pack com.pillowfresco.birdnetmqtt.sdPlugin

Output:
  com.pillowfresco.birdnetmqtt.streamDeckPlugin
